## 👨‍💻 About Me

Full Stack Developer with production experience across backend microservices, web frontend, and cross-platform mobile — spanning **NestJS, Next.js, React Native, PostgreSQL, MongoDB, Redis, and Kafka**. Comfortable owning features end-to-end: from system design and API architecture to third-party integration and deployment.
 
Currently shipping on **Higher** — a live K-pop fan platform serving **100K+ iOS/Android users** — covering a NestJS microservices monorepo, a Next.js 14 checkout site, and a React Native WebView bridge, all in a single production monorepo.
 
I operate with an **AI-first mindset** — using **Claude Code**, **Cursor**, and **GitHub Copilot** not as shortcuts, but as force multipliers: scaffolding architecture, debugging complex distributed flows, and accelerating test coverage, while always reviewing and hardening output before it ships.
